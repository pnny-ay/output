## 資料概要

### handmade_susume.pdf
- わたしがハンドメイド作家をはじめる上で考えたこと/やったこと.

### patchbased_presentation.pdf
- 卒業論文の最終発表資料. 
- 現実に基づいた物理の支配方程式をコンピュータで計算することで現実に近いCGを作成する「物理シミュレーション」の研究室.
- 卒業研究では炎に着目し, 近年需要の高い高解像度のシミュレーションを高速にするための手法を提案, 検討した.

### parachute_presentation.pdf
- 大学院での研究. 研究室は大学と同じ.
- 昼花火のシミュレーションの研究. 中でもパラシュートに発煙筒をつけたものを打ち上げる「煙竜」を対象としており, 現在はまずパラシュートの実装を進めている.
- 2020/9月時点の進捗報告資料.
- プログラムの一部を「parachute」リポジトリにて公開.
